# NETWORK+ COLLEGE PATH
The following course layout is "competancy based" and is only split into 6 major sections.
The last section is the Network+ Exam itself.
The descriptions for each competency are the course objectives.
## CompetenUnderstand concepts related to the OSI model.
Know the different networking appliances, applications, and functions, and how they differ from each other.
Understand cloud concepts and connectivity options.
Know common networking ports, protocols, services, and traffic types.
Know the different transmission media and transceivers, their use cases and how they differ from each other.
Know different network topologies, architectures, and types, their use cases, and now they differ from each other.
Given a scenario, use appropriate IPv4 network addressing.
Understand evolving use cases for modern network environments.cy 1 Networking Concepts

## Competency 2 Network Implementation
Understand characteristics of routing technologies.
In a given scenario, decide which switching technologies and features are needed and know how to install and configure them.
In a given scenario, decide which wireless devices are needed and know how to install and configure them.
Understand  important factors of physical installations.
## Copetency 3 Network Operations
Explain the purposes of different organizational processes and procedures.
Given a scenario choose and use the best network monitoring technologies.
Explain disaster recovery concepts.
Given a scenario, implement IPv4 and IPv6 network services.
Compare & contrast network access and management technologies.
## Competency 4 Network Security
Understand the importance of basic network security concepts.
Know various types of attacks and their impact to the network.
Given a scenario apply network security features, defense techniques, and solutions.
## Competency 5 Network Troubleshooting
Understand the network troubleshooting methodology
Given a scenario, troubleshoot common cabling and physical interface issues.
Given a scenario, troubleshoot common issues with network services.
Given a scenario, troubleshoot common network performance issues.
Given a scenario, choose and use appropriate tools and/or protocols to solve networking issues.
